# Engine Installation


## Overview 
The [Engine](https://github.com/enowars/EnoEngine) is the core Software glueing everything togehter, it consists of: 

 - EnoEngine
 - EnoLauncher
 - EnoFlagSink
 - EnoELK
    * Elasticsearch  
    (search engine, noSQL)
    * Logstash (ingest and transform data)
    * Kibana (webfrontend)
 - EnoMoloch
 - [ScoreBoard](https://github.com/enowars/EnoLandingPage)
